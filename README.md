# Swoolegento - Project Template

## Getting started
<ul>
    <li>Install Docksal - <a href="https://docs.docksal.io/getting-started/setup/">https://docs.docksal.io/getting-started/setup/</a></li>
    <li>Copy .docksal/docksal-local.env.sample and rename to docksal-local.env</li>
    <li>In docksal-local.env enter Magento Marketplace credentials for installing Magento and change MySQL port mapping if it will conflict with anything else you have running</li>
    <li>Run <code>fin start</code></li>
    <li>After a few minutes a Magento 2.4.5 demo store should be available at https://your-folder-name.docksal</li>
</ul>
